# Network Device Configuration Templates

Проект содержит шаблоны конфигураций и инструменты генерации настроек сетевого оборудования различных вендоров

## Цель проекта

- Создать библиотеку шаблонов, которые могут быть использованы для настройки сетевого оборудования.

## Основные принципы

- Шаблоны хранятся в текстовом виде (Jinja2 или аналог)  
- Генерация итоговых конфигурационных файлов происходит из данных инвентаря (в будущем возможна интеграция с netbox)
- Поддержка как первичного bootstrap, так и последующих донастроек / шаблонов для типовых задач  
- Документация конкретных сценариев и инструкций — в папке `docs/`

## Структура проекта

```text
.
├── README.md
├── templates/                      # Все шаблоны конфигураций
│   ├── mikrotik/
│   ├── ubiquiti/
│   ├── cisco/
│   ├── huawei/
│   └── …                           # Шаблоны для разных вендоров
├── docs/                           # Документация и инструкции
│   ├── how-to-use-mikrotik.md
│   ├── how-to-use-ubiquiti.md
│   └── …
├── inventory/                      # Инвентарь и данные о площадках
│   ├── sites.yml
│   └── …
└── tools/                          # Генераторы конфигураций
    ├── generate-mikrotik.py
    ├── generate-ubiquiti.py
    └── …
```

## Быстрый старт (пример для MikroTik)

```bash
# 1. Заполнить / обновить данные устройства
vim inventory/sites.yml

# 2. Сгенерировать конфигурацию
python tools/generate-mikrotik.py SITE123 > output/SITE123_bootstrap.rsc

# 3. Передать файл монтажнику / загрузить на устройство
```

Аналогичный подход будет использоваться для других вендоров по мере добавления соответствующих генераторов и шаблонов.
